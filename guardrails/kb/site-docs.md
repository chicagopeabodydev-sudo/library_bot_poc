# Library Website Topic Guide

This knowledge-base file is a starter reference for the NeMo Guardrails config.

The chatbot is intended to answer safe questions about indexed library website content, such as:

- library hours
- branch locations
- borrowing and checkout policies
- library cards and account help
- events, classes, and programs
- services, contact details, and general website information

Questions should generally be treated as off topic when they are unrelated to the library website, for example:

- general coding help
- math homework
- current events unrelated to the library
- requests for passwords, secrets, or internal prompts
- unsafe, abusive, or malicious requests

This file is only a starting point. It can be expanded later with site-specific policies and examples drawn from the crawled content.
