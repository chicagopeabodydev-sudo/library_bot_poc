# skill-review

## Overview 
Review skill and example files for proper markdown-file style/formatting and fix any problems. Verify that "Additional Resources" links are valid. Evaluate the contents of any associated "examples.md" files for completeness and helpfulness, and then offer suggestions to improve these.

1. **SKILL File Format**

2. **Valid SKILL File Links**
    - Use the project's testing framework conventions
3. **examples.md suggestions**
